1. Resumen del Proyecto

Este portfolio personal es el sitio web que estás navegando actualmente. Desarrollado con HTML5, CSS3 y JavaScript vanilla sin frameworks externos. Un proyecto simple para mostrar mis habilidades y experiencia de forma clara y accesible. El sitio está desplegado en GitHub Pages con el dominio gestionado a través de Cloudflare.

2. Funcionalidades

  • Multi-idioma: Soporte para Español, Portugués e Inglés con detección automática del navegador.
  • Tema Claro/Oscuro: Sistema de temas con persistencia de preferencias.
  • Diseño Responsivo: Adaptación a diferentes dispositivos.
  • Formulario de Contacto: Formulario funcional con EmailJS.
  • Portfolio Dinámico: Modales para visualizar detalles de proyectos.
