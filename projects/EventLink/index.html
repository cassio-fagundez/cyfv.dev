1. Resumen del Proyecto

EventLink es una aplicación Android nativa diseñada para centralizar y facilitar el acceso a información sobre eventos sociales, académicos y culturales. El sistema conecta a organizadores y participantes en una plataforma unificada, simplificando la publicación y el descubrimiento de eventos. Gracias a su soporte multi-idioma y sistema de validación de documentos (CI/CPF), está preparada para operar de forma escalable tanto en Uruguay como en Brasil.

2. Problema y Solución

  • Problema: La fragmentación de la información sobre eventos en regiones como la frontera de Brasil y Uruguay, que dependen de comunicaciones verbales o redes sociales poco eficientes.
  • Solución: Una aplicación Android que centraliza eventos, permitiendo a los usuarios filtrar por categorías, ver ubicaciones en tiempo real y gestionar el interés en los mismos.

3. Stack Tecnológico

  • Lenguaje: Java (JDK 11).
  • Entorno: Android Studio Ladybug (2024.2.1) | SDK 34.
  • Backend & Cloud (BaaS):
    • Firebase Authentication (v23.1.0): Gestión de usuarios y seguridad.
    • Firebase Realtime Database (v21.0.0): Sincronización de datos NoSQL en tiempo real.
  • Mapas y Geo-servicios: OSMDroid (v6.1.15) (OpenStreetMap) para una implementación de mapas de código abierto y alto rendimiento.
  • Metodología: Desarrollo Ágil (Scrum) con un enfoque Iterativo e Incremental.

4. Funcionalidades de Alto Valor

  • Internacionalización Nativa: Interfaz dinámica adaptada a Español, Portugués e Inglés, permitiendo una experiencia de usuario fluida en ambos países.
  • Mapa Interactivo Global: Visualización de eventos mediante geolocalización dinámica, permitiendo al usuario encontrar actividades sin importar su ubicación geográfica.
  • Sistema de Roles y Seguridad:
    • Participantes: Exploración y gestión de intereses.
    • Organizadores: Perfiles validados mediante documentos de identidad oficiales (CI de Uruguay / CPF de Brasil).
  • Comunicación Directa: Módulo de Preguntas y Respuestas (Q&A) con opción de anonimato y sistema de notificaciones ante cambios en los eventos.
