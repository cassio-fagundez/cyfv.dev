# Assets

Coloca aquí:

- **`icon.ico`** — Ícono del sitio (favicon y logo en el header). Se usa en la pestaña del navegador y centrado en el encabezado.
- **`photo.jpg`** — Tu foto de perfil (recomendado: cuadrado, mínimo 400×400 px).
- **`cv.pdf`** — Tu CV en PDF para el botón "Descargar CV".

El sitio funciona sin ellos: si no hay foto se muestran las iniciales "CF"; el enlace del CV puedes cambiarlo en `index.html` si lo hosteas en otro lugar.
